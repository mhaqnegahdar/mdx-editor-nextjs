# Welcome to MDX!

This is a **demo** of MDX editing and rendering.

<ConfettiButtonDemo />

## Features

* Write *markdown* with **formatting**
* Add `inline code` and code blocks
* Create [links](https://nextjs.org)
* Build lists and tables

### Code Example

```js
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
